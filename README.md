# IPL-Data-Analytics
IPL Data Analytics project using Microsoft Excel to analyze match results, team performance, player statistics, and season trends through dashboards and visualizations.
IPL Data Analytics (Excel Project)

This project focuses on analyzing Indian Premier League (IPL) data using Microsoft Excel.
The goal of this project is to extract meaningful insights about teams, players, and match outcomes through data analysis and visual dashboards.

Project Objectives

Analyze team performance across seasons

Identify top run scorers and wicket takers

Study match-winning patterns

Compare team success rates

Create interactive dashboards using Excel

Tools Used

Microsoft Excel

Pivot Tables

Charts and Graphs

Data Cleaning Techniques

Dashboard Design

Key Insights

Top-performing teams across seasons

Most consistent batsmen and bowlers

Toss impact on match results

Venue-wise performance trends

Features

Interactive Excel dashboard

Team-wise and player-wise analysis

Season-based performance comparison

Easy-to-understand visual charts

Project Files

IPL dataset (Excel)

Dashboard and analysis sheets

Learning Outcomes

Data cleaning and preprocessing

Data visualization techniques

Use of pivot tables for analysis

Dashboard creation in Excel
